This is a school project for Comp 4312 Cloud Computing.

The project includes features such as...

- High scores containing player name and score

- Multiplayer in real time using the cloud to continually send information back and forth between the server and the client.

- For scalability, players will be split into 'rooms' if there are too many people playing at once. This is to prevent the game area from being too overcrowded and to reduce stress on the player's client.

- Players can join the same 'room' as a friend through a shareable link in order to play together.
